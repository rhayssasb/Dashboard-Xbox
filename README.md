
# Dashboard de Vendas do Xbox com Excel 

Aqui você encontrará meu projeto do curso Análise de Dados com Excel e Copilot através do Bootcamp Ciencia de Dados com o Santander + Dio. 

Nessa Dashboad é possível encontrar uma pergunta de negócio:  
Qual faturamento total de vendas de plano anuais(contendo todas as assinaturas agregadas)?

Por meio de um menu de é possível selecionar a opção desejada, após a seleção será visualizada a resposta  por meio de um gráfico.
## O que foi usado?

Durante a construção da Dashboard foram utilizadas algumas importantes ferramentas:

1. Construção de uma tabela dinamica a partir dos dados fictícios fornecidos pelo material de apoio

2. Paleta de cores em Hexadecimal

3. Gráficos dinamicos 

4. Filtros visuais

Além de outras ferramentas como imagens e formas. 


## Processo

O processo foi construído a partir de um aprendizado contínuo adquirido nas seguintes aulas:

1. Método ABCDE

2. Entendendo nossa Base de Dados

3. A importância do Data View

4. Gráficos Dinâmicos e Filtros Visuais

5. Organização e Estilização da Área de Dashboard

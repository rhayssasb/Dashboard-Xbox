# Dashboard-Xbox
Nesse repositório você encontrará um dashboard de vendas feito como parte de um desafio de projeto do Bootcamp Santander de Ciencia de Dados.

A dashboard foi feita para responder a seguinte informação: TOTAL SUBSCRIPTION EA PLAY SEASON PASS de forma sazonal ou anual.

Foi usado o Excel e habilidades adquiridas por meio do curso Análise de dados com Excel e Copilot
